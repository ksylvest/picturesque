= Picturesque

Picturesque is a Ruby on Rails image resizing tool that enables simple dynamic image resizing.

== Requirements

The gem is tested with:

* Ruby on Rails 4.2.0
* Ruby 2.2.0
* JRuby

== Installation

  gem install picturesque

== Additional

  brew install imagemagick

== Configuration

  # config/initializers/picturesque.rb
  Picturesque.config do |config|
    config.finder do |params|
      Photo.find(params[:id]).url
    end
    config.metadata do |params|
      metadata = {}
      metadata['Cache-Control'] = 'public'
      metadata['Expires'] = Time.now + 3.years
      return metadata
    end
  end

== Examples

Migration:

  rails g model photo url:string

  class CreatePhoto < ActiveRecord::Migration
    def self.up
      create_table :videos do |t|
        t.string :url

        t.timestamps
      end
    end

    def self.down
      drop_table :videos
    end
  end

Model:

  class Photo < ActiveRecord::Base
    validates_presence_of :photo
  end

View:

  <%- @photos.each do |photo| -%>
  <%= picturesque_url photo, size: '480x480', format: Picturesque::Format::JPG %>
  <%- end -%>
